NEVERLOSE REST API

This API allows you to signup users, do CRUD operations on items. Finders can report the lost item and owners can change item status using Firestore Database.

Available Endpoints

Users

  • GET api/user/me - Get logged in user
  • POST api/user/signup - Signup a new user
  • POST api/user/login - logs in a user

Items

  • POST api/items/ - Create item for a user
  • PUT api/items/:itemId - Update the item by ID
  • GET api/items/ - Get items for users
  • Get api/items/owner/:itemId - Get item details by ID for owner
  • DELETE api/items/:itemId - Delete income entry

Public

  • GET api/public/items/:token - Get only few item details for public by token
  • CREATE api/public//items/:token/found/ - Create a found report for an item by finder

Reports

  • GET api/reports/item/:itemId - Get reports for items
  • PATCH api/reports/:reportId - Update the ststus of the report

Labels

  • GET api/labels/:itemId/pdf - Generate PDF label by item id.

Upload

  • POST api/upload/photo - Upload the photo

Logs

  • GET api/logs/item/:itemId - Get the item logs by ids
